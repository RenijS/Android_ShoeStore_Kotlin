# Shoe Store App
Shoe store app where user can add new item and look through list. In this app, you can see some use of Room, Navigation Component, Data Binding, RecyclerView etc. 

The app has 6 screens:

1. Login 
2. Welcome
3. Instruction
4. Shoe List
5. Add Shoe
6. Shoe Detail
